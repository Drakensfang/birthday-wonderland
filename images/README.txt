INSTRUCTIONS FOR ADDING YOUR IMAGES
====================================

Place your 9 obstacle images in this folder with the following names:

obstacle1.jpg  (or .png, .gif)
obstacle2.jpg
obstacle3.jpg
obstacle4.jpg
obstacle5.jpg
obstacle6.jpg
obstacle7.jpg
obstacle8.jpg  <- This one hides the key!
obstacle9.jpg

Tips:
- Images can be .jpg, .png, or .gif format
- They will be automatically cropped to fit square cards (80x80px)
- Use fun, colorful Alice in Wonderland themed images!
- Recommended: playing cards, teacups, mushrooms, clocks, roses, etc.
- obstacle8.jpg is special - it hides the golden key behind it

If you want to use different file formats, update the src paths in landing.html:
- Change "obstacle1.jpg" to "obstacle1.png" for example
- The extension must match your actual file

Example images you could use:
- Photos of playing cards
- Pictures of teacups
- Alice in Wonderland character images
- Colorful patterns or textures
- Anything whimsical and fun!
